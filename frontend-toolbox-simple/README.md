## 🧠 How to use this repo

> Keep this README as your **frontend database**. Every time you find a useful tool, add it to the right section.

**Rating system**

| Mark | Meaning |
|---|---|
| ⭐⭐⭐⭐⭐ | Essential. Use often. |
| ⭐⭐⭐⭐ | Very useful. Keep close. |
| ⭐⭐⭐ | Good, but not always needed. |
| 🧪 | Need to test. |
| 🧊 | Archive / maybe later. |

**Status tags**

`using` · `testing` · `learn` · `saved` · `replace?` · `favorite`
